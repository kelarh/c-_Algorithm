# 关于计算机图形学的一些算法实现

## 环境配置
- Visual studio code
- Install .NET Core SDK
- Install Extension of C# div Kit
## Dotnet 基础命令
```bash
dotnet new winforms -n myProject # 创建winform项目
```
```bash
dotnet run # 运行
```
